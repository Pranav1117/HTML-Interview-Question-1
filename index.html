Day 1

  • Difference between span and div:

    1. Div is an block level tag and span is inline tag
    2. Div always start on new line and it will take whole width of line and span takes only required width and it does not take unnecessary width
    3. you can chnge width and height of block element and u cannot change width and height of inline element

  • Difference between Class and ID:

    1. Class and id's are selctors and are used to give name to element, using id and class we can taget a particular or multiple element.
    2. Ids are unique and classes are not, we can apply multiple classes to any element but we cannot apply same value of any id to different elemnet. In one file there should be just one value of id
    3. we denote id with hash and classs with dot.

  • What are self closing tags?

    • self tags are tags which does not required closing tag to be terminated

  • How many types of heading are there?

    • There are 6 types of heading from h1 to h6, Most important is h1 and least important is h6. Basically h1 and h2 used for main heading.

    • Strong tag is used to show importance of text and em tag is used show element to highlight in italic format(they are prefer to use over bold and italic tags). While using em tag it will change tone of the voice of the visitor

  • What are the attributes?

    • attributes gives additional info about tags, it should be written in lowercase letter only, it should be given in the openig tag always. Multiple attributes can be used in particular tag.
